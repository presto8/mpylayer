mpylayer is a python package to easily control **mplayer** (and in the [future](ToDoList.md) **mencoder**) in python, using a _pythonic OO syntax_.

A quick example:

```
>>> mp = mpylayer.MPlayerControl()
>>> mp.loadfile('/path/to/some_music.mp3')
>>> mp.volume = 100
```
